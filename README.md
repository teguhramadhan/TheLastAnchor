# TLA_DEVHUB v0.1.4
### [PROJECT: THE LAST ANCHOR] | [INTERNAL_DEVELOPMENT_TOOL]

Repositori ini berisi aplikasi web internal (DevHub) yang berfungsi sebagai pusat dokumentasi teknis, database crafting, dan Game Design Document (GDD) untuk proyek game survival vertikal **The Last Anchor**.

Aplikasi ini dirancang sebagai pendamping saat bekerja di Unity 6, memberikan akses cepat ke balancing data dan referensi mekanik tanpa harus membuka file dokumen statis.
